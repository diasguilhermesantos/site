Componentes de Computadores e Grandezas Elétricas

1. Componentes de Computadores

1.1 Componentes Internos

1.1.1 CPU (Unidade Central de Processamento)

A CPU (Central Processing Unit) é o cérebro do computador, responsável por processar e executar instruções de programas e do sistema operacional.

1.1.2 Armazenamento

O armazenamento se refere a dispositivos como HDDs e SSDs, que guardam os dados e programas do computador de forma permanente.

1.1.3 Memória

A memória RAM (Memória de Acesso Aleatório) é usada para armazenar temporariamente os dados que estão sendo processados pelo sistema.

1.1.4 Fonte de Alimentação

A fonte de alimentação converte a energia elétrica da tomada para os componentes do computador, garantindo a estabilidade e proteção contra picos de tensão.

1.2 Componentes Externos

1.2.1 Teclado

O teclado é um dispositivo de entrada que permite digitar comandos, textos e interagir com o sistema operacional.

1.2.2 Monitor de Vídeo

O monitor exibe informações gráficas e vídeos, permitindo que o usuário interaja visualmente com o computador.

1.2.3 Mouse

O mouse é um dispositivo apontador que permite navegar e interagir com interfaces gráficas do computador.

1.3 Interfaces de Entrada e Saída

1.3.1 Interfaces Sem Fio

Interfaces sem fio, como Wi-Fi e Bluetooth, permitem comunicação sem cabos entre dispositivos e redes.

1.3.2 Áudio

Interfaces de áudio incluem conectores para fones de ouvido, caixas de som e microfones.

1.3.3 Rede (Ethernet)

A interface de rede Ethernet permite conexões cabeadas para acesso à internet e redes locais.

1.3.4 VGA

O VGA é um padrão antigo de conexão de vídeo usado para monitores e projetores.

1.3.5 HDMI

O HDMI transmite áudio e vídeo digital de alta qualidade para monitores e TVs.

1.3.6 USB

O USB (Universal Serial Bus) é usado para conectar diversos dispositivos, como teclados, mouses, pen drives e câmeras.

2. Grandezas Elétricas

2.1 Potência Elétrica

A potência elétrica mede a quantidade de energia consumida ou fornecida por um dispositivo, geralmente expressa em watts (W).

2.2 Tensão Elétrica

A tensão elétrica, medida em volts (V), representa a diferença de potencial elétrico entre dois pontos.

2.3 Corrente Elétrica

A corrente elétrica, medida em amperes (A), indica o fluxo de elétrons em um circuito.
